# Tech Stack

## Frontend
- **Framework:** React v19
- **Build Tool:** Vite v7
- **Routing:** wouter
- **State Management:** TanStack Query (React Query)
- **Styling:** Tailwind CSS v4, PostCSS
- **UI Components:**
  - Radix UI (Primitives)
  - Lucide React (Icons)
  - Embla Carousel
  - Recharts (Charts)
  - Sonner (Toasts)
  - Vaul (Drawers)
  - Framer Motion (Animations)
- **Forms & Validation:** React Hook Form, Zod

## Backend
- **Runtime:** Node.js
- **Framework:** Express
- **Authentication:** Passport.js (Local Strategy)
- **Session Store:** Connect PG Simple / MemoryStore
- **Email:** Resend
- **AI/ML:** Ollama (Local LLM integration)

## Database
- **Database:** PostgreSQL
- **ORM:** Drizzle ORM
- **Schema Validation:** Drizzle Zod

## Mobile
- **Cross-Platform:** Capacitor v8

## DevOps & Tooling
- **Language:** TypeScript
- **Package Manager:** npm
- **Runner:** tsx
- **Linting:** TypeScript (tsc)
